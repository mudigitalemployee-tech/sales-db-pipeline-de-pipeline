## Pipeline Architecture

```
Source CSV
    ↓
[Bronze Layer]  — Raw ingestion, no changes
    ↓
[Silver Layer]  — Clean, dedup, type-cast, null-handle, PII-mask
    ↓
[Gold Layer]    — Business aggregations (KPIs for dashboard)
    ↓
[Report]        — MuSigma HTML report + ZIP delivery
```

## Running the Pipeline

```bash
pip install -r requirements.txt
# Run manually
python3 pipeline/airflow_dag.py

# Run DQ checks
python3 pipeline/dq_checks.py --data_path data/transformed.csv
```
